# Test Import Repository

A sample Python repository for testing the GitHub repository import feature.

## Overview

This repository contains a simple Python application that demonstrates basic functionality including:
- Greeting function
- Time/date utilities
- Basic calculations

## Files

- `main.py` - Main Python application with sample functions
- `requirements.txt` - Python dependencies
- `README.md` - This documentation file

## Installation

1. Clone this repository:
```bash
git clone https://github.com/7inh/test_import_repo.git
cd test_import_repo
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the sample application:
```bash
python main.py
```

Or import the functions in your own code:
```python
from main import greet, get_current_time, calculate_sum

# Use the functions
message = greet("World")
current_time = get_current_time()
total = calculate_sum([1, 2, 3, 4, 5])
```

## Dependencies

See `requirements.txt` for a complete list of dependencies. Main packages include:
- requests - HTTP library
- pandas - Data analysis library
- numpy - Numerical computing library
- pytest - Testing framework
